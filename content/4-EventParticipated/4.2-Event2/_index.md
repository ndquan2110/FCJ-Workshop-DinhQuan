# Report on “Event Meetup”


### List of Speakers
Nguyễn Huỳnh Sơn

### Key Topics

#### SLA and Monitoring

Speaker: **Nguyễn Huỳnh Sơn**

In the session titled "SLA and Monitoring – From SLA to Monitoring: What Really Matters," Mr. Nguyễn Huỳnh Sơn introduced the fundamental concepts and practical experience in building, monitoring, and ensuring the quality of modern IT systems on the AWS cloud platform. The presentation emphasized that successfully deploying a system is only the first step, while maintaining system stability, monitoring performance, and ensuring a good user experience are the key factors in delivering high-quality services.

#### Inside the Exam: AWS Cloud Practitioner

Speaker: **Ngô Lê Tấn Huy**

In the seminar "Inside the Exam: AWS Cloud Practitioner – A Strategic Roadmap to Conquering the AWS Cloud Practitioner Exam," Mr. Ngô Lê Tấn Huy provided an overview of the AWS Certified Cloud Practitioner (CLF-C02) certification. He also shared an effective learning roadmap and exam preparation strategies. The session not only helped participants understand the exam structure but also provided essential knowledge about Cloud Computing and the Amazon Web Services (AWS) ecosystem.

#### Securing Your Web Apps with AWS Security Agent

Speaker: **Nguyễn Tuấn Thịnh**

In the seminar "Securing Your Web Apps with AWS Security Agent," Mr. Nguyễn Tuấn Thịnh shared practical knowledge and experience in building a comprehensive security process for web applications on AWS. The presentation focused on applying Artificial Intelligence (AI) together with AWS security services to automate vulnerability detection, risk assessment, and support developers in building more secure applications.

### Highlights

The three seminars focused on important topics within the Amazon Web Services (AWS) ecosystem from different perspectives.

SLA and Monitoring helped participants understand how to monitor systems and ensure service quality.
Inside the Exam: AWS Cloud Practitioner provided fundamental cloud computing knowledge and guidance for earning the AWS Cloud Practitioner certification.
Securing Your Web Apps with AWS Security Agent focused on web application security, DevSecOps, and the use of Artificial Intelligence (AI) to automate security processes.

Overall, the presentations were highly practical and helped participants improve their knowledge of Cloud Computing, system operations, cybersecurity, and career development in Information Technology.

### Knowledge Gained

After attending the three AWS seminars, I gained valuable knowledge about Cloud Computing, system monitoring, and application security. These sessions not only strengthened my theoretical understanding but also provided practical insights that align with current industry trends.

First, I developed a better understanding of Service Level Agreements (SLA) and their importance in ensuring service quality. I learned that building a system is only the beginning; maintaining system stability, availability, and performance is equally important. I also understood the role of Monitoring and learned how tools such as Amazon CloudWatch can be used to monitor system performance, collect logs, configure alarms, and detect issues promptly. One of the most important lessons was that monitoring user experience is just as important as monitoring infrastructure resources.

In addition, the AWS Cloud Practitioner session gave me a comprehensive overview of cloud computing and the AWS ecosystem. I learned about cloud service models such as IaaS, PaaS, and SaaS, as well as deployment models including Public Cloud, Private Cloud, and Hybrid Cloud. I also became familiar with core AWS services such as Amazon EC2, Amazon S3, Amazon RDS, AWS Lambda, Amazon VPC, AWS IAM, and Amazon CloudWatch, understanding their functions and common use cases. Furthermore, I gained insight into the AWS Certified Cloud Practitioner exam structure, key knowledge domains, and effective preparation strategies.

### Application to Study and Future Career

From an academic perspective, the knowledge gained from these seminars will help me better understand how to design, deploy, and operate cloud-based systems. I can apply these concepts in courses such as Cloud Computing, Software Development, DevOps, System Administration, and Information Security.

When working on academic or personal projects, I can utilize AWS services such as Amazon EC2 for server deployment, Amazon S3 for data storage, Amazon RDS for database management, and Amazon CloudWatch for system monitoring. My understanding of SLA and Monitoring also enables me to evaluate service quality, monitor application performance, and identify issues during software development.

Moreover, the knowledge gained about the AWS Certified Cloud Practitioner certification provides me with a solid cloud computing foundation and a clearer learning roadmap. I plan to prepare for this certification and later pursue advanced certifications such as AWS Solutions Architect Associate, AWS Developer Associate, or AWS SysOps Administrator Associate, all of which are highly recognized in the IT industry.
